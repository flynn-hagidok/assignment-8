Name: Hero Apps

Description
Hero Apps is a modern React-based application marketplace where users can browse, search, install, and uninstall apps. The platform provides detailed information about each application, including ratings, downloads, reviews, and app size. Users can manage their installed apps and sort them based on ratings for a better experience.

Technologies Used
React
React Router
Tailwind CSS
DaisyUI
Recharts
React Toastify
Local Storage API
JavaScript (ES6+)
Vite
